# Product Overview

This is a fitness coaching platform for personal trainers to manage clients, workouts, nutrition plans, and scheduling.

## Core Features

- **Client Management**: Admin dashboard for managing students/clients with profiles, progress tracking, and status monitoring
- **Workout Management**: 
  - Video library with exercise demonstrations
  - PDF workout plans
  - Structured workout cards (fichas de treino) with exercises, sets, reps, and rest periods
  - Exercise tracking and progress logging
- **Nutrition Plans**: Meal planning system with customizable nutrition programs
- **Scheduling**: Professional agenda system with time blocks, appointments, and availability management
- **Payment Tracking**: Subscription and payment management for clients
- **Progress Tracking**: Client progress photos, metrics, and evolution tracking

## User Roles

- **Admin/Trainer**: Full access to manage clients, create content, view analytics, and manage schedules
- **Student/Client**: Access to assigned workouts, nutrition plans, scheduling, progress tracking, and community features

## Tech Context

Built as a full-stack TypeScript application with Supabase backend, deployed on Vercel.
